## 2. 공문서 세분류 (공문 하위 유형)

현재 "공문" 유형 하나로 통합 구현. `content_requirements`로 목적 지정 가능.
별도 유형 분리 시 아래 참고.

| 유형 | 용도 | 특징 |
|------|------|------|
| 시행문 | 결정사항 시행 통보 | "~시행합니다" 어미, 시행일자 필수 |
| 통보문/안내문 | 정보 전달, 일정 안내 | "~알려드립니다", "~안내합니다" |
| 조회문 | 자료/의견 요청 | "~회신하여 주시기 바랍니다" |
| 회신문 | 조회에 대한 답변 | "귀 기관 ~호 관련하여" 시작 |
| 의뢰문 | 업무 협조 의뢰 | "~의뢰합니다", "~요청합니다" |
| 보고문 | 상급기관에 보고 | "~보고합니다" |

## 3. 결재/품의 문서

| 유형 | 용도 | 비고 |
|------|------|------|
| 기안문(품의서) | 내부 의사결정 요청 | 결재선: 담당→팀장→부서장 |
| 지출결의서 | 예산 집행 승인 요청 | 금액, 계정과목, 산출근거 필수 |
| 구매요청서 | 물품/용역 구매 요청 | 품명, 수량, 예산, 납기 |
| 출장신청서 | 출장 승인 요청 | 기간, 목적지, 사유, 경비 |
| 휴가신청서 | 휴가 승인 요청 | 유형, 기간, 사유 |

## 4. 보고서 세분류

| 유형 | 용도 | 주요 섹션 |
|------|------|-----------|
| 출장보고서 | 출장 결과 보고 | 출장개요→방문내용→성과→후속조치 |
| 감사보고서 | 내부감사 결과 | 감사개요→지적사항→개선권고→조치계획 |
| 경영실적보고서 | 분기/연간 경영성과 | 경영목표→실적→분석→차기계획 |
| 월간업무보고서 | 월별 업무 현황 | 금월실적→익월계획→이슈사항 |
| 현장점검보고서 | 현장 점검 결과 | 점검개요→점검결과→조치사항 |

## 5. KAMCO 특화 문서

한국자산관리공사 고유 업무(자산관리, 채권관리, 국유재산)에 특화된 문서 유형.

| 유형 | 용도 | 주요 섹션 |
|------|------|-----------|
| 자산매각공고문 | 국유/공공자산 매각 공고 | 매각대상→입찰조건→일정→유의사항 |
| 입찰공고문 | 조달/용역 입찰 공고 | 사업개요→참가자격→제출서류→일정 |
| 채권관리보고서 | 부실채권 정리/관리 현황 | 채권현황→정리실적→회수율→계획 |
| 국유재산관리보고서 | 국유재산 관리 현황 | 재산현황→활용실적→관리계획 |
| 타당성분석서 | 사업/투자 타당성 분석 | 사업개요→시장분석→재무분석→위험분석→결론 |
| 감정평가의뢰서 | 자산 감정평가 의뢰 | 대상물건→의뢰목적→기한→특이사항 |

## 7. 문서 형식 공통 규칙 (행정안전부 공문서 작성 규정)

### 경어체
- 합쇼체: "~합니다", "~바랍니다", "~주시기 바랍니다"

### 계층 번호 체계
- **일반형**: 1. → 가. → 1) → 가)
- **로마형** (기획안/사업계획서): Ⅰ. → 1. → 가. → 1)
- **회의록형**: 1. → ○ → -

### 문서 종결
- 본문 마지막: "끝."
- 붙임 있을 때: "붙임  1. (문서명) 1부.  끝."

### 본문 시작 관용구
- 관련 근거 있을 때: "1. 관련: ○○부 ○○과-제○○호(YYYY.MM.DD.)"
- 직접 시작: "1. (주제)에 대하여 다음과 같이 ..."
