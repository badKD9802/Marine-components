### Q2. 성장과정 (1,500자)

#### 전략: "통계학 → NLP 연구 → Agentic AI 실전" — 3개 전환점 서사

**구조 (3단, 1,500자):**

```
[전환점 1] 통계학 석사에서 NLP에 빠지다 (400자)
  - 부산대학교 통계학과 석사 과정
  - 석사 논문: "불균형 텍스트 분류" — SMOTE, B-SMOTE, ADASYN 기법 비교
  - "데이터에서 숨겨진 패턴을 찾는 것"에 매료 → 특히 텍스트라는 비정형 데이터에 통계적 방법론을 적용하는 것에 흥미
  - 깨달음: "좋은 데이터와 올바른 방법론이 모델의 성능을 결정한다"
  ↓
[전환점 2] KAMCO 프로젝트에서 "아키텍처 재설계"를 주도하다 (600자)
  - 애자일소다 선임연구원으로 KAMCO 그룹웨어 AI 도입 프로젝트 참여
  - 초기: LangGraph 기반 40개 노드 파이프라인 → 기능 추가 시 라우팅 복잡화, 다중 요청 처리 불가
  - 이 한계를 분석 → ReAct(Reasoning + Acting) 패턴을 연구
  - 직접 아키텍처 재설계를 제안 → OpenAI Function Calling 기반 자율형 Agent로 전면 전환
  - 18개 도구, 363개 자동화 테스트, 평균 응답시간 3.9초 → 프로덕션 배포
  - 깨달음: "문제의 근본 원인을 찾으면, 작은 팀원이라도 시스템 전체를 바꿀 수 있다"
  - 이것이 삼성의 "도전" 문화와 연결 — 기존 방식에 안주하지 않고 더 나은 구조를 찾는 태도
  ↓
[전환점 3] 개인 프로젝트에서 "기술을 가치로 전환"하다 (500자)
  - 아버지의 선박부품 B2B 사업 → AI를 접목할 수 있겠다는 생각
  - 프론트엔드/클라우드 배포 경험 없이, AI를 활용하여 독립적으로 해결
  - FastAPI 5개 모듈 + pgvector RAG 챗봇 + 관리자 대시보드 → 1인 기획~배포
  - 깨달음: "기술을 아는 것"과 "기술로 비즈니스 가치를 만드는 것"은 다르다
  - 삼성리서치에서 연구가 제품에 탑재되어 수십억 사용자에게 전달되는 것 = 이 경험의 극대화
```

**핵심 메시지:**
- "나는 **문제를 만나면 근본 원인을 찾고, 기존 방식을 뒤집을 수 있는 사람**"
- "학문(통계) → 실무(Agent/RAG) → 사업(풀스택 AI 서비스)"으로 확장해온 여정
- 각 전환점에서 얻은 **깨달음이 삼성전자의 가치와 연결**되도록 설계

**주의사항:**
- 1,500자 여유 있음 → **에피소드를 생생하게** 서술
- "성장과정"이지만 **가치관/태도 형성** 중심으로 (연대기 X)
- "가상인물도 포함 가능"이라 했지만, **실제 경험이 훨씬 강력**
- 삼성의 **"도전과 열정"** 문화와 자연스럽게 연결

#### 활용할 기존 자료:

  - 이 한계를 해결하는 것이 바로 Agentic AI — 사용자의 의도를 파악하고 자율적으로 기능을 선택·실행
  ↓
[역량 1] Agentic AI 설계·구현 → "자율적 문제 해결 Agentic AI 개발" 직결 (350자)
  - KAMCO 프로젝트에서 ReAct Agent 전담 설계·구현
  - LLM이 18개 업무 도구를 자율 선택·조합하는 구조 설계
  - OpenAI Function Calling 기반 도구 레지스트리, SSE 실시간 스트리밍, 중복 호출 방지
  - 363개 자동화 테스트(오타/구어체 포함), 평균 응답시간 3.9초, 프로덕션 배포
  - → 삼성리서치의 핵심 수행업무 "Agentic AI 기술 연구 및 개발"에 즉시 기여 가능
  ↓
[역량 2] RAG + Fine-Tuning → "개인화 응답 서비스 솔루션" 매칭 (300자)

  - → 삼성 가우스를 활용한 개인화된 응답 서비스, 갤럭시 AI 고도화에 기여 가능
  ↓
[마무리 — 삼성 제품과의 연결] (200자)
  - 갤럭시 AI가 "사용자가 기능을 선택하는 단계"에서 "AI가 자율적으로 판단·실행하는 단계"로 진화하는 데 기여하고 싶음
  - 삼성리서치의 Agentic AI 연구 + 삼성 가우스 + 갤럭시 하드웨어의 결합이 이 진화의 핵심
